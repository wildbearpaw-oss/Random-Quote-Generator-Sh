# Random Quote Generator — Генератор случайных цитат

## Информация об авторе
**Имя и фамилия:** Шашина Полина 
**Дата создания:** Май 2026

## Описание

Это простое GUI‑приложение на Python, которое генерирует случайные цитаты из предопределённого списка. Приложение сохраняет историю сгенерированных цитат, позволяет фильтровать их по автору и теме, а также сохраняет историю в JSON‑файл.

## Функциональность

* Генерация случайной цитаты по нажатию кнопки.
* Отображение истории сгенерированных цитат.
* Фильтрация истории по автору или теме.
* Сохранение истории в файл `history.json` при каждом добавлении новой цитаты.
* Загрузка истории при запуске приложения.

## Как использовать

1. Запустите файл `main.py`.
2. Нажмите кнопку **«Сгенерировать цитату»**, чтобы увидеть случайную цитату.
3. Введите имя автора или тему в соответствующие поля и нажмите **«Применить фильтры»**, чтобы отфильтровать историю.
4. История автоматически сохраняется в файл `history.json`.

## Зависимости

* Python 3.x
* Стандартные библиотеки: `tkinter`, `random`, `json`, `os`

## Примеры использования (тесты)

1. **Тест генерации цитаты:**
   * Нажмите кнопку «Сгенерировать цитату».
   * Убедитесь, что цитата отображается в окне приложения.
   * Проверьте, что цитата появилась в списке истории.

2. **Тест сохранения истории:**
   * Сгенерируйте несколько цитат.
   * Закройте приложение.
   * Откройте приложение снова.
   * Убедитесь, что история загрузилась из файла `history.json`.

3. **Тест фильтрации:**
   * Введите имя автора (например, «Шекспир») в поле «Фильтр по автору».

   * Убедитесь, что в истории отображаются только цитаты этого автора.

## Примеры использования

### Пример 1: Генерация цитаты
1. Запустите приложение.
2. Нажмите кнопку «Сгенерировать цитату».
3. В окне появится случайная цитата, например:
“Знание — сила”
— Фрэнсис Бэкон
Тема: Философия

4. Цитата автоматически добавится в список истории внизу окна.

### Пример 2: Фильтрация по автору
1. Сгенерируйте несколько цитат (например, 5–6).
2. В поле «Фильтр по автору» введите «Шекспир».
3. Нажмите «Применить фильтры».
4. В списке истории останутся только цитаты Уильяма Шекспира (если они есть в истории).

### Пример 3: Фильтрация по теме
1. Сгенерируйте несколько цитат.
2. В поле «Фильтр по теме» введите «Мотивация».
3. Нажмите «Применить фильтры».
4. В списке истории останутся только цитаты на тему мотивации.

### Пример 4: Комбинированная фильтрация
1. Введите «Леннон» в поле «Фильтр по автору».
2. Введите «Жизнь» в поле «Фильтр по теме».
3. Нажмите «Применить фильтры».
4. Если есть цитата Джона Леннона на тему жизни, она отобразится в истории.

### Пример 5: Проверка сохранения истории
1. Сгенерируйте 3–4 цитаты.
2. Закройте приложение.
3. Откройте приложение снова.
4. Убедитесь, что история загрузилась из файла `history.json` и отображается в списке.

## Валидация ввода

Приложение проверяет корректность ввода при фильтрации:
* Если оба поля (автор и тема) пустые, появится предупреждение: «Введите автора или тему для фильтрации».
* Поиск выполняется без учёта регистра (можно вводить «шекспир» вместо «Шекспир»).
* Фильтрация показывает цитаты, содержащие введённую подстроку (например, поиск «филос» найдёт «Философия»).
